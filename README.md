# Quiz Master

## SkillCraft Technology Internship - Task 3

A modern and interactive Quiz Game Application built using HTML, CSS, and JavaScript. The application supports both Single Player and Multiplayer modes with room creation and joining functionality.

## Features

- 🎯 Single Player Quiz Mode
- 👥 Multiplayer Quiz Mode
- 🆔 Create Room with Unique Room ID
- 🔗 Join Room using Room ID
- 📋 Copy Room ID Button
- ⏱️ 15-Second Timer for Each Question
- 📊 Progress Bar
- ✅ Instant Answer Validation
- 🏆 Final Score Display
- 🔄 Restart Quiz
- 📱 Fully Responsive Design
- 🎨 Modern UI with Gradient Theme

## Technologies Used

- HTML5
- CSS3
- JavaScript
